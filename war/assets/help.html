ROUTEMARKS

Please have a look at the video tutorial given in about page. In this section the functionalities of each screen is explained.

Home screen

Helps to check the safety of the route based on the start and stop location given by the user. The route is considered as safe when you see lesser marks on the route. By default only those marks which are created on the current date will be displayed. But user can choose following options for more details.
a) Yesterday
b) Last week
c) Last month
d) Last year
e) All
To get a tabular safety report, click on Get report button.

Report screen

Report screen will display the number of marks which are created in the given bounds. Values of the bounds are decided by the start and stop locations given by the user in the home screen. Mark counts are displayed in a tabular format for current day, yesterday, last week, last month, last year and all.

Community screen

When a user lands on the community page, the room location will be defaulted to the country on which he/she located. All the marks which are created in that country will also be displayed. And new marks will be displayed as and when they are created. User can choose a different room he/she likes. When a new location is entered, all the marks which are created in that location will be displayed to him/her. And when new marks are created, that will also be presented to the user. Users can like, dislike or star any marks. Star is used in a situation when she/he finds the mark very useful.

Mark screen

Mark page is also having a field which accepts any place name. By default the map in the page will be centered with the current location of the user; provided he has enabled browser location. User can choose his place from the field where she/he wants to create the mark. In order to create a mark, first of all, user has to click on a particular point on the map. Then a red mark will be displayed on the map. User can remove the red mark by a mere right click on it. As a second step to create a mark, user has to click on the red mark which will pops up an information window. The window will have drop down boxes for category, transport mode, reason/nature and severity. The values in each of the select boxes are given below.
a) Category
  1. Accident
  2. Natural disaster
b) Transport mode (Accident)
  1. BiCycle
  2. Car
  3. Motor Cycle
  4. Public Transport
  5. SUV
  6. Truck
  7. Train
c-1) Reason (Accident)
  1. Over speed
  2. Reckless driving
  3. Sharp Curve
  4. Traffic rule violation
  5. Wrong message board
  6. Ice road
  7. Bad road
  8. Unknown
c-2) Reason (Natural disaster)
  1. Avalanche and Landslides
  2. Earthquake
  3. Hurricane or Typhoon
  4. Tornado
  5. Tsunami
  6. Volcanic Eruptions
d) Severity
  1. Critical
  2. Major
  3. Minor

Based on the cateogry, transport mode, nature/reason and severity user can choose any values. User can also add some meaningful comment in the description field. Upon submitting the button user will be navigated to next screen in the same window. Just to ensure that the user is not a robot, he/she has to fill a basic mathematical formula and complete the process by clicking submit button which will create a new marker. When the mark is created he will also get notified in the page.

Account screen

There are two ways in which user can reach account page. Either by clicking on the login/username link or directly selecting the account tab. If the user is not logged in, the page will be rendered with various open authentication service providers. User can choose any of these services and login to RouteMarks. When he/she logs in, the page will display a green card which contains all of the user activities, points, etc.

About screen

About screen has a video tutorial on how to use this application.
