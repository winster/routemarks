Please have a look at the video tutorial given in about page. In this section the features of each section of the screen is explained.

Search

Helps to check the safety of the route based on the start and stop location given by the user. The route is considered as safe when you see lesser marks on the route. Default result will have the marks which have been created for the last year. But user can choose following options for more details from the blue drop down box right below View MarksM heading.
a) Today
b) Yesterday
c) Last week
d) Last month
e) Last year
f) All
Also get a tabular report by button.

Report screen

Report screen will display the number of marks which are created in the given bounds. Values of the bounds are decided by the start and stop locations given by the user in the home screen. Mark counts are displayed in a tabular format for current day, yesterday, last week, last month, last year and all.

Community

Messages from the community are displayed to the right of the page. Default location will be the current country of the user. All the marks which are created in that country will be displayed and new marks will be added to the list as and when they are created. User can choose a different location to see the marks which are created in that location. Users can authenticate each mark by clicking on any of the following three buttons.
a) Eye witness
b) I concur
c) Fake information

Map

Map displays the RouteMarks along with the route. A mark can be an accident location or a natural disaster affected area. Marks will be displayed as red spots on the map. User can view details of a mark by clicking on it. The then opened info window will display the parameters that user has saved while creating the mark along with the image/video. In order to create a RouteMark, user has to click on the map which makes the map to zoom in to the desired level. Now user has to click on the exact location where he wants to create the mark. Then a red mark will be displayed on the map. User can remove the red mark by a mere right click on it. If user clicks (left click) on the red mark, an information window will pop up. The info window has drop down boxes for category, transport mode, reason/nature and severity. The values in each of the select boxes are given below.

Based on the cateogry, transport mode, nature/reason and severity user can choose any values. User can also add some meaningful comment in the description field. Continue operation by selecting next button . User can share any video(youtube/vimeo) or image url or even upload an image to the application. Click . Just to ensure that the user is not a robot, fill in the basic mathematical formula and complete the process by clicking submit button which will create a new ROUTEMARK. When the mark is created user will get notified via a message.

Login / View Account

By clicking on the ID Card link, a login dialog will pop up which has various open authentication service providers listed. User can choose any of these services and login to RouteMarks. When the user logs in, the page will display a green card which contains all of the user activities, points, etc.
